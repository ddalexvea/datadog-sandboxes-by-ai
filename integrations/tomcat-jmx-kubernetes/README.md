# Tomcat JMX Monitoring on Kubernetes

**Full Sandbox Repository:** [datadog-sandbox-tomcat-jmx-kubernetes](https://github.com/ddalexvea/datadog-sandbox-tomcat-jmx-kubernetes)

## Overview

This sandbox demonstrates how to configure JMX monitoring for Apache Tomcat running on Kubernetes with the Datadog Agent.

## What's Included

- Kubernetes manifests for Tomcat deployment
- JMX integration configuration
- Datadog Agent setup with JMXFetch
- Custom metrics collection examples

## Use Cases

- JMX integration setup on Kubernetes
- Tomcat performance monitoring
- Custom JMX metrics collection
- Autodiscovery configuration for JMX
