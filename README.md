# Datadog Sandboxes Repository

This repository contains sandbox environments for reproducing and demonstrating various Datadog Agent configurations, integrations, and issues.

## Creating New Sandboxes

When creating a new sandbox, please use the standard template:

**Sandbox Template:** https://github.com/ddalexvea/datadog-sandbox-readme-template

## Structure

Each sandbox is organized by category in its own directory with a README.md placeholder that links to the full sandbox repository.

Browse the folders above to explore sandboxes by category:
- `integrations/` - Integration-specific sandboxes
- `apm/` - APM and tracing sandboxes
- `aws/` - AWS-specific deployments
- `kubernetes/` - Kubernetes-related sandboxes
- `logs/` - Log collection sandboxes
- `otel/` - OpenTelemetry sandboxes
- `bugs/` - Bug reproduction environments
- `autoscaling/` - Autoscaling and HPA sandboxes
- And more...
